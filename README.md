# Виселица (игра)
Реализация первого проекта из [курса](https://zhukovsd.github.io/java-backend-learning-course/) Сергея Жукова по Java Backend 
разработке. <br>
Интерфейс игры консольный.
### Правила и принцип игры
Загадывается слово. Загаданное слово выводится в консоль, где каждая буква заменена на звездочку. 
Игрок предлагает букву, которая может входить в это слово. Если такая буква есть в слове, 
то звездочка заменяется на соответствующую букву — столько раз, сколько она встречается в слове.
За каждую ошибку к висельнику дорисовывается часть тела. Максимальное количество ошибок - 6. 
### Функционал приложения и меню консольного интерфейса
1. При старте, приложение предлагает начать новую игру или выйти из приложения.
2. При начале новой игры, случайным образом загадывается слово, и игрок начинает процесс по его отгадыванию.
3. После каждой введенной буквы в консоль выводится счётчик ошибок, текущее состояние виселицы.
4. По завершении игры выводится результат (победа или поражение) и предложение начать новую игру
или выйти из приложения.
